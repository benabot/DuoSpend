# Skill — Monétisation DuoSpend

## Position retenue

- Freemium.
- 1 projet gratuit.
- Achat unique non-consommable pour projets illimités.
- Pas de publicité.
- Pas d'abonnement tant qu'une décision projet contraire n'a pas été actée.

## Règles produit

- Le paywall ne doit pas bloquer l'usage avant la création du 2e projet.
- La version gratuite doit exposer le coeur de valeur.
- Le texte de paywall doit être factuel, sans dark pattern.
- Le restore purchase doit être simple à trouver.

## Points de vigilance

- cohérence entre `COMMERCIAL.md`, `SETUP_STOREKIT.md` et le code ;
- cohérence entre Product ID, prix affiché et entitlements ;
- cohérence entre fonctionnalités Pro annoncées et réellement présentes.
